Features

recent messages dataset

CopyCat has access to the dataset of recent messages posted in a channel, and uses this dataset for detection of duplicate messages.

various message types

CopyCat detect duplicates not only amongst text messages but messages containing emojis, messages containing urls, and messages containing uploaded files of any type.

24/7 availability

CopyCat is always available, and always working to detect duplicates.
